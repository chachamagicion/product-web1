
# Lotto Number Generator

## Overview

This project is a simple web application that generates random lottery numbers. It's built using modern web technologies, including HTML, CSS, and JavaScript, with a focus on creating a clean, visually appealing, and user-friendly experience.

## Features

*   **Lottery Number Generation:** Generates a set of 6 unique random numbers between 1 and 45.
*   **Modern UI:** A clean and modern user interface with a visually balanced layout, custom fonts, and a subtle background texture.
*   **Responsive Design:** The application is fully responsive and works on both desktop and mobile devices.

## Current Plan

*   **HTML Structure:** Create the basic HTML structure with a title, a container for the lottery numbers, and a button to trigger the generation.
*   **CSS Styling:** Style the page with a custom color palette, fonts, and a visually appealing layout.
*   **JavaScript Logic:** Implement the JavaScript logic to generate and display the unique lottery numbers.
